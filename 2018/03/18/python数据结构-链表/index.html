对于node类只需关注两点：value 和 next

方法：

  • is_empty() 链表是否为空
  • add(item) 链表头部添加元素
  • append(item) 链表尾部添加元素
  • insert(index, item) 指定位置添加元素
  • delete(index) 删除指定位置节点
  • remove(item) 删除某个数值节点
  • search(item) 查找节点是否存在
